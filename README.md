# DharBlu HIZ Portfolio Website

Personal branding website milik Muhammad Altaf Adharta / DharBlu HIZ.

Website ini dibuat untuk menampilkan identitas digital, pengalaman, portfolio, serta social media dalam tampilan modern bertema ocean blue UI.



## Features

- Responsive Design
- Modern UI/UX
- Smooth Scroll Navigation
- Active Navbar Indicator
- Reveal Animation on Scroll
- Portfolio Filter
- Video Modal Popup
- Featured Works Section
- Social Media Integration
- Mobile Friendly Navigation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
